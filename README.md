# Machine Learning & Pattern Recognition Notebooks

Collection of Jupyter notebooks for machine learning, clustering, classification, and explainable AI.

## Setup

```bash
pip install -r requirements.txt
jupyter notebook
```

## Contents

- credit_risk_synthetic_prediction - 214211R.ipynb
- kmeans_dbscan_gmm_analysis - 214211R.ipynb
- Lab sheet/ - Clustering, Decision Trees, SVM, and XAI tutorials

## Topics

Clustering (K-Means, DBSCAN, GMM), Classification (Decision Trees, SVM), Explainable AI (LIME, DICE), Credit Risk Prediction, Customer Behavior Analysis
